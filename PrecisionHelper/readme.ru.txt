Precision Helper 1.0.4
======================

Precision Helper это инструмент для создания и управления проектами справки
в формате Microsoft Compiled HTML Help (CHM).

Программа специализируется на организации существующих файлов html, xml,
сценариев, изображений и других ресурсов так, чтобы автор справки имел
наилучшее представление своего проекта.

Precision Helper работает с исходным форматом Microsoft HTML Help projects (HHP) и позволяет создавать конечную справку в форматах CHM, WebHelp, PDF и одиночного документа html. 

Precision Helper предназначен для опытных разработчиков и авторов, но дружественная среда позволяет создавать проекты справки также начинающим пользователями. 

Отсутствие режима WYSIWYG и редактора исходного текста - осознанная особенность Precision Helper. На рынке программного обеспечения есть много качественных редакторов, которые можно выбрать как дополнение к программе. Precision Helper не хочет скрывать или управлять любой частью исходного файла вместо автора. Программа оставляет автору свободу выбора в использовании любимых программ редактирования. Для этого Precision Helper может быстро и удобно открыть определенный внешний редактор с выбранным файлом раздела.

Precision Helper распространяется БЕСПЛАТНО. Для ознакомления прочтите license.ru.txt 


Что нового?
-----------
- перевод на русский язык
- перевод на польский язык

- незначительные усовершенствования и устранение  незначительных ошибок


Системные требования:
--------------------
Аппаратные средства:
  IBM-совместимый ПК
  256 MB RAM
   10 MB свободного места на диске

Операционная система: 
  Microsoft Windows 2000/XP/Vista

Дополнительные требования
  Microsoft HtmlHelp Workshop (для компиляции в формат CHM)
  Microsoft Internet Explorer 6.0+ (для предпросмотра разделов)


Установка:
-------------
Запустите инсталлятор и следуйте командам мастера установки. Для установки под Windows Vista вы должны обладать правами администратора.

Портативная версия может быть просто распакована в желаемую папку и использована запуском файла PrecisionHelper.exe

Примечание:
  В системе должен быть установлен Microsoft Html Help Workshop, чтобы правильно компилировать справку в формат Microsoft CHM. Фактически нужны только несколько dll-файлов этой программы, но Microsoft не рекомендует устанавливать и регистрировать их вне этой программы. Мастер установки автоматически спросит Вас о ее загрузке и установит Microsoft Html Help Workshop, если он не установлен на вашей системе.
