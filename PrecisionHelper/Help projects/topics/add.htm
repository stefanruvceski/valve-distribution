Add

Opis

Add prozor omogucuje dodavanje novog ventila u postrojenje.

Polja

ID polje zahteva broj koji vec nije iskoriscen kao ID nekog drugog ventila vec unetog u postrojenje.

Name polje zahteva tekstualni naziv datog ventila.

Type polje omogucava odabir tipa ventila, opcije su Tip1, Tip2, Tip3.

Add dugme izvrsava dodavanje ventila sa zadatim ID ako vec ne postoji u postrojenju.

Exit dugme zatvara Add prozor i vraca se na glavni prozor tacnije World position tab.
