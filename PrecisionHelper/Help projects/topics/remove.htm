Remove

Opis

Remove prozor omogucuje uklanjanje ventila u postrojenje po njegovom ID.

Polja

ID polje zahteva broj koji se kao ID nekog ventila u postrojenju.

Remove dugme izvrsava uklanjanje ventila sa zadatim ID ako postoji u postrojenju.

Exit dugme zatvara Remove prozor i vraca se na glavni prozor tacnije World position tab.
