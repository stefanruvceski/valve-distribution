Table filter tab

Opis

Table filter tab ima dva prikaza podataka i deo za odabir odredjenih filtera. Prvi prikaz je tabelarni i on se menja u skladu sa odabirom filtera. Drugi prikaz je tekstualni, od dinamicki ispisuje sva stanja ventila koja se nalaze u aplikaciji. Deo za odabir filtera sadrzi odabir nacina, krijerijuma i vrednosti filtriranja.

Precica za Table filter tab je dugme 'T' na tastaturi.

Tabelarni prikaz ventila

ID je olona tabelarnog prikaza u kojoj se ispisuju svi ID-evi ventila.

Name je kolona tabelarnog prikaza u kojoj se ispisuju nazivi datog ventila.

Type je kolona tabelarnog prikaza u kojoj se ispisuju tipovi datog ventila.

State je kolona tabelarnog prikaza u kojoj se ispisuje poslednje stanje datog ventila.


Teksualni prikaz

Polje sadrzi informacije o svakom ventilu koji se nalazi u aplikaciji, zajedno sa svim stanjima od pocetnog trenutka monitoringa do trenutnog. Dinamicki dodaje stanja ventila kako ona stizu u aplikaciju.


Filter

Filter by je polje u kojem se bira po cemu ce se filtrirati tabela. Opicije su ID, Name,Type.

Filter je polje u kojem se bira kriterijum po kom ce se filtrirati tabela. Opicije su less than, grater then,equal to.

Value je polje u koje se unosi vrednost koja ce se koristiti zajedno sa gore pomenuta dva filtera da bi se uspesno isfiltrirala tabela.
